YT Playlist Watcher

GitHub Actionsが10分ごとにプレイリストを監視
新着動画をPush通知 → Google Driveに自動ダウンロード

Push通知の設定
有効にすると新着動画・ダウンロード完了を通知します
▼ GitHubシークレット SUBSCRIPTIONS_JSON に貼り付けてください
受信した通知
セットアップ手順
  1. このページを開いて「通知を有効にする」をクリック
  2. 表示されたJSONを SUBSCRIPTIONS_JSON シークレットに貼り付け
  3. PLAYLISTS_JSON に監視プレイリストを設定
  4. GitHub Actions が自動で10分ごとに実行されます
PLAYLISTS_JSON の例
[
  {
    "id": "PLxxxxxxxxxxxxxxxx",
    "title": "お気に入りPL",
    "autoDownload": true,
    "driveFolder": "GoogleDriveのフォルダID"
  }
]
